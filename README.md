##  How It Works

- The user enters a **postfix expression** where each operand or operator is separated by a space.
- Each click on the **Next** button performs **one step** of the postfix evaluation algorithm:
  - Operands are pushed onto the stack.
  - Operators pop two operands, apply the operation, and push the result back.
- The **Previous** button allows navigating back to earlier steps.
- Once evaluation is complete, the **final result** is displayed.
